# ARCHITECTURE_LITERAL.md (P0)

## Purpose
This document describes only what exists and runs in MetaBlooms OS.

## Literal structure
MetaBlooms OS consists of:
- Canonical root discipline (stable paths and entrypoints)
- Deterministic pipelines
- Validators and gates
- Append-only ledgers

## Non-features
MetaBlooms does not generate goals, does not self-modify rules without explicit deltas, and does not possess cognition. The LLM is an untrusted generator subject to validation.
