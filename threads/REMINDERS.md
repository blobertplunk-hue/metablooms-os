# Hanging Thread Reminder Contract (P0)

## Surfaces
- BOOT: list unresolved threads + cadence due
- SHIP: list unresolved threads + cadence due
- WEEKLY_REVIEW: list all threads with cadence=weekly or overdue

## Output Format (suggested)
- thread_id
- state
- intent (one line)
- blocking_requirements (one line)
- next_valid_actions (one line)
