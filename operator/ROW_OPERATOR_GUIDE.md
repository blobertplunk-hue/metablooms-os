# ROW Operator Guide (v1)

ROW = Routed Operational Workflow.
A ROW is a routing contract: intent → which pipelines activate, which are automatic, which are deferred.

## Invocation patterns
- Start a ROW: CREATE_HTML (TEKS 3.8A, Google Sites)
- Start a ROW: FIX_HTML (embed blank on iPad)
- Start a ROW: PRACTICE_KATAS (CSS overflow)
- Start a ROW: AUDIT_EMBEDS (these 20 links)
- Start a ROW: INCORPORATE_DELTAS (apply this zip)

## Fail-closed
If intent is missing/ambiguous → ROW_INTENT_DISCOVERY (no artifact production beyond a short intent menu).
